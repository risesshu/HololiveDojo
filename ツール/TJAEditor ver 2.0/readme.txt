[概要]
太鼓の達人をシミュレーションするソフト
"太鼓さん次郎"の譜面ファイルであるTJAファイルを
編集するためのテキストエディタです。

[使用方法]
TJA File Editorを開いて、メニューまたはツールバーから
新規作成を選んで初期状態にしてください。
あとは自由に譜面を作成するだけです。

[機能]
・太鼓さん次郎自動実行
メニュー→太鼓さん次郎→太鼓さん次郎の場所設定を選んで、
太鼓さん次郎がある場所を選んでください。
そうするとメニュー→太鼓さん次郎→太鼓さん次郎起動
もしくはF5キーを押すことで、
現在編集している譜面を太鼓さん次郎で開くことができます。
ただし、ファイルを一回保存しなくてはいけません。


・ショートカットキー
CTRLキーと以下のキーを同時に押すことでさまざまな文字を挿入することができます。

    B : #BPMCHANGE
    G : #GOGOSTART
    O : #GOGOEND
    U : #MEASURE
    S : #SCROLL
    D : #DELAY
    T : #SECTION
    R : #BRANCHSTART
    A : #BRANCHEND
    N : #N
    E : #E
    M : #M
    L : #LEVELHOLD

・ツールメニューにあるツールの使い方
	・文字の置換
	 置換前の文字と置換後の文字を入力して、ボタンを押すと、
	 選択された範囲の文字が置換されます。

[注意]
同じフォルダに入っているFooty2.dllは移動させないでください。
